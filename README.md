
# Application Review System

This is an application review system that can be used by any firm for managing candidates.


## Features

- Django REST Framework
- Add Multiple Candidates
- Add Multiple Experiences , Educations and links to a single candiate
- View Resume without downloading
- Detail view of every candidate

## Tech Stack

**Client:** React.js, HTLM, CSS

**Server:** Python , Django
